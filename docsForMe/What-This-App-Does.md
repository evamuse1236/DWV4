# What This App Does

Deep Work Tracker helps students build consistent study habits and helps teachers/coaches monitor progress.

There are two sides:
- Student side: daily check-in, planned work, learning tasks, diagnostics, reading.
- Admin side: assign work, review requests, manage queues, and track progress.

The main flow is simple:
1. Student checks in.
2. Student does planned work.
3. Student asks for mastery checks.
4. Admin reviews and approves/advances progress.

Things that matter most:
- Major progress status is tracked in one main place (`studentMajorObjectives`).
- Diagnostics and pass/fail logic are decided on the backend.
- AI features must return machine-readable blocks so UI can use them safely.
